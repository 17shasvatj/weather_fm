# weather_fm
Mini Weather Foundation model trained on subset of ERA5 data

Variables used for now:

2m_temperature, mean_sea_level_temperature, 10m_u_component_of_wind, 10m_v_component_of_wind, 2m_dewpoint_temperature, surface_pressure, total_precipitation

Coordinates: [50, -130, 24, -60] (North, West, South, East)
Region: Continental US
Resolution:  1° × 1°
Time: 6-hourly
Years: 2015-2020
